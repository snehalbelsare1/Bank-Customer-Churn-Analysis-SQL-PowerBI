# Power BI Dashboard

This folder contains the Power BI dashboard file for customer churn analysis.

---

## Report Structure

### 1. Executive Dashboard

- Core KPIs (Churn Rate, Revenue at Risk, Active Members)
- Churn distribution by geography
- Churn distribution by age group
- Churn by product count
- Interactive slicers

### 2. Customer Profile Analysis

- Balance vs Credit Score visualization
- Tenure-based segmentation
- Demographic churn patterns

### 3. Business Insights and Strategy

- Key churn drivers identified
- Business problem statements
- Recommended retention strategies
- Expected business impact

The dashboard is structured to simulate an executive-level business performance review.
