# SQL Analysis

This folder contains SQL scripts used for data preparation, transformation, and analytical computation.

---

## Workflow Covered

### Data Staging
- Created structured staging table from raw dataset

### Data Cleaning
- Removed identifier columns not relevant for analysis
- Validated null values and data consistency

### Feature Engineering
- Created Age Group segmentation
- Created Credit Category segmentation

### KPI Computation
- Calculated churn rate
- Calculated active member percentage
- Computed average balance and credit score
- Estimated revenue at risk

### Segmentation Queries
- Churn by Geography
- Churn by Tenure
- Churn by Product Count
- Churn by Activity Status

All queries follow a structured analytical workflow similar to a production reporting environment.
