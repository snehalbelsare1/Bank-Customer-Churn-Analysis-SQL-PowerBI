	Bank Customer Churn Analysis & Retention Strategy

* Project Overview
1. This project analyzes customer attrition patterns in a retail banking dataset to identify key churn drivers and propose data-driven retention strategies.
2. The analysis was performed using SQL Server for data preparation and KPI modeling, and Power BI for interactive dashboard development.

* Business Objective 
Customer churn directly impacts revenue and long-term profitability.

* This project aims to:
1. Calculate churn rate and core business KPIs
2. Identify high-risk customer segments
3. Analyze churn behavior across geography, tenure, age, and product ownership
4. Provide actionable retention strategies

* Key Insights

1.Overall churn rate stands at 20.37%.
2.Inactive members demonstrate significantly higher churn probability.
3.Customers with 1 product show higher exit rates.
4.Senior customers contribute disproportionately to total churn.
5.Germany exhibits the highest churn rate among regions.

* Business Recommendations

1.Launch engagement campaigns targeting inactive members.
2.Promote bundled product offerings to increase product penetration.
3.Implement targeted retention programs for senior customers.
4.Deploy region-specific marketing initiatives in high-churn geographies.
